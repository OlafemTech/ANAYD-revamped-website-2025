Navbar Test Page

This page demonstrates the responsive navbar component with mobile menu functionality.

Features

  • Responsive design for desktop and mobile
  • Dropdown menus for desktop navigation
  • Mobile slide-out menu with animations
  • Nested dropdown menus in mobile view
  • Active page highlighting
  • Scroll behavior with background changes

Implementation

The navbar is implemented as a component that can be loaded into any page using the component loader system. It uses Tailwind CSS for styling and vanilla JavaScript for interactions.

To add this navbar to any page, simply include the component loader script and add a div with id="navbar-container".
